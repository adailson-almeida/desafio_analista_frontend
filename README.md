# Gerenciador de Chamados

* Aplicação web responsiva para gerenciamento de chamados.

## Instalação | Execução

1. Clonar o projeto acessando o repositório público no GitHub:
https://github.com/adailson-almeida/desafio_analista_frontend.git

2. Executar o comando **npm install** para instalar todas as depêndencias
e pacotes necessários

3. Executar o comando **ng serve** ou **npm start** para build do projeto:
http://localhost:4200/

## Tecnologias Utilizadas

1. Angular 21.0.3
2. Node 24.11.1
3. TypeScript
4. PrimeNG
5. HTML5
6. CSS3

* A escolha do **Angular 21** (versão atual) foi por querer explorar as novas abordagens 
e a questão do standalone também;

* Optei pela biblioteca **PrimeNG** pela variedade de componentes e seus designers agradáveis que acredito enriquecerem a UX do projeto.

## Percentual de código gerado por IA e Desenvolvedor

* Estimo que **30%** do código foi gerado por **IA (Gemini)** e **70%** pelo **Desenvolvedor**.

**Obs.** O uso de IA no projeto foi moderado ao máximo, foram feitas pesquisas e consultas afim de ***sanar dúvidas*** de ***compatibilidade entre versões do framework e da biblioteca***, reforçar ***boas práticas*** do Angular e consultar classes utilitárias entre outros detalhes otimizando o tempo gasto no projeto.
